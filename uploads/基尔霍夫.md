## 一、为什么需要基尔霍夫？

你肯定学过欧姆定律：\(U = I R\)。这很漂亮，但只能对付一个电阻和一个电源。给你两个电阻串并联，用欧姆定律加加减减还能算。可如果电路像蜘蛛网一样：好几个电源，电阻交叉连接，，欧姆定律就不太行——因为你不知道哪条支路的电流是多少，电压怎么分配。

这时候就需要基尔霍夫。专门用来分析这种复杂网络。这两条规则不依赖电阻的具体值，只依赖电流和电压的“流动逻辑”。

---

## 二、第一个规则：电流定律（KCL）

**通俗叫法：节点电流定律。**  
核心一句话：**在任何一个节点（三根或更多导线汇合的点），流入的电流总和等于流出的电流总和。**

相当于什么？
就像水管的三通接头：如果你往接头里灌水，从三通流出去的水加起来必须等于灌进来的水。水不会凭空消失，也不会凭空产生。电也是一样：电荷不会在节点堆积。

**数学写法**（但你不用背，要会想）：  
\(\sum I_{\text{in}} = \sum I_{\text{out}}\)  
或者把所有电流都看作带正负号的代数和：\(\sum I = 0\)（规定流入为正，流出为负）

**举个例子**：  
一个节点上连着三根线。一根线有2A电流流入，另一根线有3A电流流入，那么第三根线必须是流出5A。  
反过来，如果一根线流入5A，另一根流出2A，那第三根必须流出3A。

**怎么用**：  
- 先假设每条支路电流的方向（随便猜，猜错了结果会是负数，负数表示真实方向相反）。  
- 在每一个节点列一个方程。  
- 注意：有n个节点，你最多只能列n-1个独立方程（因为最后一个方程是前面几个的和）。

## 三、第二个规则：电压定律（KVL）

**通俗叫法：回路电压定律。**  
核心一句话：**沿着任意一个闭合回路走一圈，电压升的总和等于电压降的总和。**

这又是什么意思？
想象你在一个游乐园里爬楼梯：你从地面出发，上5级台阶到一个平台，再上3级到更高处，然后下7级回到地面。你上升的总台阶数（5+3=8）等于下降的总台阶数（7？不对，还差1？那说明你算错了）。实际上，你最终回到了地面，所以总上升高度必须等于总下降高度。电压也一样：你从一点出发，沿着回路走回原点，电势的升高和降低相互抵消，总和为零。

**数学写法**（同样先理解意思）：  
\(\sum U_{\text{rise}} = \sum U_{\text{drop}}\)  
或者代数和：\(\sum U = 0\)（规定电压降为正，电压升为负）

**例子**：  
一个最简单的回路：电池（电压12V）和一个电阻（2Ω），电流I。  
从电池负极出发，走到正极：电压升12V（这是电源提供的）。  
再从电池正极经过电阻回到负极：电压降 \(I \times 2\)。  
根据KVL：12V - 2I = 0，所以I=6A。这就是欧姆定律本身，其实欧姆定律是KVL的特例。

**更复杂的例子**：  
两个电池和三个电阻组成的回路。你沿着回路走，遇到电池正到负是电压降？注意方向。规定一个绕行方向（顺时针或逆时针），然后：
- 电阻上的电压：如果电流方向和绕行方向相同，就是电压降（取正）；相反就是电压升（取负）。
- 电池上的电压：从负极到正极是电压升（取负，因为你在等式另一边），从正极到负极是电压降（取正）。  
但新手最容易弄反。我教一个不容易错的办法：**不管正负号，你列方程时，先把所有元件的电压按照绕行方向加起来，结果等于0**。你只要每次写好：+ 还是 -。练习几次就熟了。

## 四、基尔霍夫怎么用？（完整步骤）

假设有一个电路，有两个电源和三个电阻，连成两个回路。要求每条支路的电流。

**步骤1：标出所有支路电流，并假定方向。**  
每个分支上画一个箭头。比如 \(I_1, I_2, I_3\)。随便猜，没关系。

**步骤2：用KCL在节点上列方程。**  
比如节点上有 \(I_1\) 流入，\(I_2\) 和 \(I_3\) 流出，那么 \(I_1 = I_2 + I_3\)。

**步骤3：用KVL在回路上列方程。**  
选几个独立的回路（每个回路至少有一条新支路）。沿着回路走一圈，把电压升降加起来等于0。

**步骤4：解方程组。**  
你会有几个方程（节点方程数+回路方程数 = 支路数），正好求解每个电流。

**关键点**：如果算出来的电流是负数，说明你刚开始假设的方向反了，没关系，大小对了就行。

## 五、用手比划一个具体例子（别怕，手算给你看）

电路：  
左边一个12V电池（正极在上，负极在下）。从电池正极出发，接一个 \(R_1 = 2\Omega\)，然后到了节点A。从节点A分出两条路：  
- 一条路经过 \(R_2 = 3\Omega\) 回到电池负极（这是第一条回路）。  
- 另一条路经过 \(R_3 = 4\Omega\)，然后接到另一个电池 \(V_2 = 6V\)（正极在上），最后也回到电池负极。  
看起来有点乱？我画个文字图：

```
        R1=2Ω
   +---/\/\/--- A
   |            |
 12V           / \
   |           R2 R3  3Ω和4Ω
   |           |  |
   +-----------|  +--- 6V ---+
                \ /          |
                 B           |
                 +-----------+
```

（B点就是下面公共点）

我们设：  
\(I_1\) 从12V正极流出，经过R1到A。  
在A点，\(I_1\) 分成两路：\(I_2\) 经过R2到B，\(I_3\) 经过R3再到6V电池正极，然后从6V负极出来到B。

**KCL在A点**：\(I_1 = I_2 + I_3\)  
**KCL在B点**：\(I_2 + I_3 = I_1\)（和上面一样，所以只有一个独立节点方程）

**KVL回路1**（左边那个圈：从12V负极出发，上到正极，经过R1到A，再经过R2回到负极）：  
绕行方向顺时针。  
先碰到12V电池：从负极到正极是电压升，按绕行方向看是？我们规定绕行方向和电流方向可以不同，但要小心。我换一种稳妥方式：  
电压升总和 = 电压降总和。  
升：电池12V。  
降：R1上电压 \(I_1 \times 2\)，R2上电压 \(I_2 \times 3\)。  
所以 \(12 = 2I_1 + 3I_2\)。

**KVL回路2**（右边那个圈：从A经过R3，经过6V电池，再经过R2回到A）：  
绕行方向顺时针（从A→R3→6V正→6V负→R2→A）。  
电压升：6V电池？这里小心：从6V正极到负极是电压降，但我们的路径是？从A走R3到6V正极，然后从6V正极到负极（这是电池内部，电压降6V），然后从负极走R2回到A。所以总电压升？我们来列总和=0的方法：  
从A点出发，经过R3：假设电流\(I_3\)方向是从A到6V正极，那么R3上的电压是 \(I_3 \times 4\)，这个电压方向与绕行方向相同（从A到6V正），所以是电压降？我乱了对吧。不搞晕你，我们直接代公式：  
对于回路2，绕行方向设为顺时针（A→R3→6V正→6V负→R2→A）。  
列出各项电压（电压降为正，升为负）：  
- 经过R3：电流\(I_3\)方向与绕行方向一致，所以电压降 \( + I_3 \times 4\)。  
- 经过6V电池：从正到负是电压降？不对，我们的路径是从6V正极（碰到）然后走到负极，这是顺着电池的降的方向，所以是电压降 \(+6V\)。  
- 经过R2：电流\(I_2\)方向是从A到B？注意：回路2中经过R2的方向是从B到A（因为我们从6V负极出发，走R2回到A）。而之前设\(I_2\)是从A到B。所以在R2上，实际电流方向与绕行方向相反，因此电压升，即 \(- I_2 \times 3\)。  
总和：\( +4I_3 + 6 - 3I_2 = 0\)，即 \(3I_2 - 4I_3 = 6\)。

现在我们有三个方程：  
(1) \(I_1 = I_2 + I_3\)  
(2) \(12 = 2I_1 + 3I_2\)  
(3) \(3I_2 - 4I_3 = 6\)

解：由(2)代入(1)：\(12 = 2(I_2+I_3) + 3I_2 = 5I_2 + 2I_3\)  
与(3)联立：  
\(5I_2 + 2I_3 = 12\)  
\(3I_2 - 4I_3 = 6\)  
解一下：第一个乘2得 \(10I_2+4I_3=24\)，加第二个方程：\(13I_2 = 30\) → \(I_2 \approx 2.3077A\)  
代入得 \(4I_3 = 3I_2 - 6 = 6.9231-6=0.9231\) → \(I_3 \approx 0.2308A\)  
\(I_1 = I_2+I_3 = 2.5385A\)。

验算一下：R1电压=5.077V，R2电压≈6.923V，R3电压≈0.923V，回路1：12V = 5.077+6.923=12V。回路2：0.923V (R3) + 6V = 6.923V (R2上但反向? 实际R2电压6.923V，方向与绕行相反，所以 -6.923 + 0.923 + 6 =0。正确。

## 六、为什么这个办法有效？基本道理

- **KCL** 基于电荷守恒：电荷不会在节点消失，所以流入等于流出。  
- **KVL** 基于能量守恒：你绕着回路走回原点，电势不能变，所以升高的电势必须等于降低的。

这两个定律加上元件的伏安特性（比如欧姆定律），就足够解出任何线性电路。

## 七、使用方法

1. **不要怕假设方向**：错就错，负数告诉你反了，数量对就行。  
2. **先标节点，再选回路**：节点选三个以上导线交点。回路选那些相互独立的（至少包含一个新支路）。  
3. **检查方程个数**：支路数b，节点数n，独立回路数 = b - n + 1。如果你列的回路数超过这个，会有冗余方程。  
4. **如果电源是电流源**：电流源的电压未知，需要多设一个变量。  
5. **可以先用戴维宁简化部分电路**，再用基尔霍夫，两者不冲突。

---

## 八、与戴维宁的关系

戴维宁定理其实就是基尔霍夫定律的一个应用：通过基尔霍夫算出开路电压和短路电流，然后得到等效电阻。基尔霍夫是更底层的“宪法”，戴维宁是“快捷方式”。  
简单电路用欧姆定律，复杂电路用基尔霍夫，要反复分析同一个负载变化就用戴维宁。

---

**一句话总结基尔霍夫**：  
两个简单规则——电流在节点“进出平衡”，电压在回路“升降平衡”——就能把乱糟糟的电路变成可解的方程组。按步骤做，别怕负号，你就能算出每条路的电流。
