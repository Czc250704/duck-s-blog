**一、认识电容器及分类与图形符号**

**什么是电容：**
```
电容器简称电容，任何两个彼此绝缘又相互靠近的导体都可以构成电容器，这两个导体称为电容器的两个极板，中间的绝缘材料称为电容器的介质。![](/uploads/picture/电工图片/dianrongyuanli.png)
```
**电容的特性：**
```
电容器最基本的特性的是能够储存电荷
```
**电容分类：**
-   **按结构分**：固定电容器（无极性）、可变电容器、微调电容器。
-   **按介质分**：电解电容（有极性）、瓷片电容、涤纶电容、聚丙烯电容（CBB）等。
**图形符号**：需重点区分 **无极性电容**和 **有极性电容**。如下图所示：![电容器](/uploads/picture/电工图片/series.png)


**二、掌握电容器的主要参数及公式**

-   **标称容量**：常用单位微法（μF）、纳法（nF）、皮法（pF）。换算关系为 1μF = 10³nF = 10⁶pF。
-   **额定电压**（耐压值）：是电容长期可靠工作的最高电压，使用时实际电压绝不能超过此值。
-   **允许误差**：常用±5%（J级）、±10%（K级）、±20%（M级）等表示。
-   **绝缘电阻与漏电流**：衡量介质绝缘好坏，理想情况应无穷大。
-   **公式**：
```
电容器所储存的电荷量与两极板间的电压的比值是一个常数，称为电容器的电容量，简称电容，用字母C表示。用公式表示为![原理](/uploads/picture/电工图片/dianrongyuanli.png)，[公式](/uploads/picture/电工图片/dianronggongshi.png)。
平行板电容器的电容量与电介质的介电常数及极板面积成正比，与两极板间的距离成反比，用公式表示为[原理](/uploads/picture/电工图片/pingxingbandianrongqi.png)，[公式](/uploads/picture/电工图片/pingxingbangongshi.png)。
```

**三、学会识别电容器的标志及型号**

-   **型号**：[型号图](/uploads/picture/电工图片/dianrongxinghao.png)
-   **直标法**：直接印出容量和耐压，如 “100μF 25V”。
-   **数码法**：用三位数字表示，单位pF。前两位为有效数字，第三位是10的指数，如 **“104” = 100000pF = 100nF = 0.1μF**。
-   **色标法**：类似色环电阻的读法，在电容上读色环或色点。

**四、掌握电容器的基本特性**

-   **“隔直通交”**：阻断直流电，允许交流电通过。
-   **“通高频、阻低频”**：容抗 \(X_C = \frac{1}{2\pi fC}\)，频率越高、容量越大，容抗越小。

**五、会用万用表检测电容器（实践重点）**

-   **1. 漏电电阻的测量（测好坏）**
    选万用表 **R×1k** 或 **R×10k** 档，测无极性电容两引脚。正常时，指针应**迅速回摆至无穷大**附近；若阻值固定很小，说明漏电击穿。

-   **2. 电解电容极性的判别**
    测两次正向、反向漏电电阻。**阻值大的那一次**，黑表笔接的是**正极**，红表笔接的是负极。这是因为正向漏电小、电阻大。

-   **3. 电容容量的粗略判断**
    万用表调零后，两表笔同时接触电容两脚，观察指针摆动幅度。对同一档位，**摆动幅度越大，容量越大**。测量电解电容时，需注意极性对应。

**六、了解电容器的选用原则**

-   **容量与耐压**：容量按电路要求选，耐压需留有余量，一般选为工作电压的**1.5~2倍**。
-   **高频/低频应用**：
    -   **高频电路**（去耦、谐振）：选瓷片、云母电容。
    -   **低频电路**（耦合、旁路、滤波）：选纸介、涤纶或电解电容。
    -   **电源滤波**：必须用大容量电解电容。
    -   **正负电极不能接反**：电解电容、钽电容接反极易损坏甚至爆炸。
