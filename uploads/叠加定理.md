## 一、叠加定理解决什么问题？

你已经学了基尔霍夫定律，能解任何复杂电路，但要列好几个方程，解起来有点麻烦，尤其电源多的时候。

现在假设你有一个电路，**里面有两个或更多独立电源**（比如一个12V电池和一个6V电池，或者一个电池加一个电流源）。你想知道某一条支路上的电流或者某两个点之间的电压。

叠加定理说：**在多电源的线性电路中，你可以一次只让一个电源单独工作，算出它产生的电流/电压；然后换成下一个电源单独工作，再算一次；最后把所有电源单独作用的结果加起来，就得到所有电源共同作用的结果。**

通俗讲：就像你同时被两个人推，你最终的位置变化等于第一个人单独推你的位置变化加上第二个人单独推你的位置变化。互不干扰。

---

## 二、核心规则（很重要）

在让某一个电源“单独工作”时，其他电源要“置零”：
- **电压源** → 用一根导线代替（短路）
- **电流源** → 断开（开路）

为什么？因为电压源短路后两端电压为零，不影响其他电源；电流源开路后电流为零，也不影响。这样就把其他电源“关掉”了。

**注意**：受控源（比如压控电压源）不能置零，必须保留在原位。叠加定理对受控源也成立，但受控源始终处于“工作”状态，不能关掉。

## 三、一步一步教

用一个具体例子。假设电路有两个电压源和一个电阻网络，我们要求某个电阻上的电流 \(I\)。

**电路描述**：  
左边一个 \(V_1 = 10V\) 电池，右边一个 \(V_2 = 5V\) 电池，中间有三个电阻：\(R_1=1\Omega\) 接在 \(V_1\) 正极，\(R_2=2\Omega\) 接在中间，\(R_3=3\Omega\) 接在 \(V_2\) 正极，所有负极连在一起作为公共地。我们要求 \(R_2\) 上的电流 \(I\)（方向假设从左到右）。

**步骤1：让 \(V_1\) 单独工作，\(V_2\) 置零（短路）**  
把 \(V_2\) 换成一根导线。此时电路变成：10V电池，\(R_1\)，\(R_2\)，\(R_3\) 现在两端都被那根导线连到地了，所以 \(R_3\) 和那根导线并联（其实 \(R_3\) 被短路了）。计算这时 \(R_2\) 的电流 \(I_1\)：  
\(R_3\) 短路，所以电流不经过 \(R_3\)。电路简化成：10V 电池 → \(R_1=1\Omega\) → \(R_2=2\Omega\) → 回到电池负极。总电阻 \(1+2=3\Omega\)，总电流 \(10/3 \approx 3.333A\)。这个电流就是 \(I_1\)，方向从左到右（和假设方向一致）。

**步骤2：让 \(V_2\) 单独工作，\(V_1\) 置零（短路）**  
把 \(V_1\) 换成导线。此时电路变成：5V电池，\(R_3=3\Omega\)，\(R_2=2\Omega\)，\(R_1=1\Omega\) 现在两端被导线短接（因为 \(V_1\) 短路，\(R_1\) 一端接地，另一端也通过导线接地，所以 \(R_1\) 被短路）。  
电流路径：5V正极 → \(R_3\) → \(R_2\) → 短路导线（原 \(V_1\) 位置）→ 回到5V负极。注意 \(R_1\) 被短路，不参与。总电阻 \(R_3+R_2=3+2=5\Omega\)，电流 \(5/5=1A\)。这个电流方向：从5V正极流出，经过 \(R_3\) 和 \(R_2\) 回到负极。在 \(R_2\) 上，电流方向是从右向左（因为正极在右端）。我们原来假设 \(I\) 的方向是从左到右，那么 \(V_2\) 单独作用时产生的电流与假设方向相反，所以 \(I_2 = -1A\)。

**步骤3：叠加**  
总电流 \(I = I_1 + I_2 = 3.333A + (-1A) = 2.333A\)（方向从左到右为正）。

这样就完成了。你不需要列基尔霍夫方程组。

## 四、叠加定理的注意事项（容易错的地方）

1. **只能用于线性电路**：电阻、电容、电感、独立源、受控源可以，但二极管、三极管（非线性）不行。  
2. **只能叠加电流和电压**：功率不能叠加！因为功率是电流的平方或电压的平方，平方后交叉项不为零。如果你分别算两个电源单独作用时的功率，加起来不等于总功率。  
3. **每次只留一个独立源**，其他独立源严格置零（电压源短路，电流源开路）。  
4. **受控源永远保留**，不能置零。  
5. **方向问题**：叠加时一定要带正负号。你可以在电路图上标出每一个电源单独作用时产生的电流/电压的实际方向，然后根据参考方向决定正负。

---

## 五、为什么叠加定理成立？

线性电路满足**齐次性**和**可加性**。每一个电源单独作用时的响应（电压或电流）与该电源的大小成正比。多个电源同时作用时，总响应就是各个电源单独作用的响应之和。这本质上是数学上的线性方程组的性质。

基尔霍夫定律列出的方程是线性方程组，叠加定理就是线性系统的一个必然结论。

## 六、叠加定理什么时候用？

- 电源数量不多（2-3个），但电路结构有点乱，你不想解方程组。  
- 你想知道**某个特定电源对某条支路贡献了多少**，比如分析哪个电源是主要影响。  
- 在交流电路中，不同频率的电源可以分别叠加（因为不同频率不会产生平均功率，但瞬时值仍可叠加）。  
- 作为理解戴维宁定理和诺顿定理的基础（戴维宁的开路电压就是所有电源共同作用的结果，但你可以用叠加定理去算它）。

如果电源很多（比如5个以上），叠加定理要算5次，反而比基尔霍夫麻烦。这时候用基尔霍夫或者节点法更快。

---

## 七、和戴维宁、基尔霍夫的关系

- **基尔霍夫**是通用方法，任何线性电路都能解，但要列方程组。  
- **叠加定理**是技巧，把多电源问题分解成多个单电源问题，避免解方程组。  
- **戴维宁定理**是化简方法，把复杂电路从负载两端看进去等效成一个电压源串电阻。而求开路电压时，你完全可以用叠加定理分别计算每个电源的贡献再加起来。

三者互补：  
当你面对一个电路，如果只需要一条支路的响应，且电源不多，叠加定理很直观；  
如果需要反复换负载，用戴维宁；  
如果电源多且不想用叠加（因为要算很多次），直接上基尔霍夫。

## 八、一个带电流源的例子（快速演示）

电路：一个 \(I_s = 3A\) 电流源（向上），一个 \(V_s = 6V\) 电压源（正极在上），两个电阻：\(R_1=2\Omega\) 和 \(R_2=4\Omega\) 串联。电流源与 \(R_2\) 并联，电压源与 \(R_1\) 串联，最后 \(R_1\) 和 \(R_2\) 串联成一个回路？具体不画了，说重点：

要求 \(R_2\) 上的电压 \(U\)。

- 先让电流源单独工作：电压源短路。电流源3A流过 \(R_2\) 和 \(R_1\) 并联？不对，看连接方式。假设正确连接后，算出 \(U_1 = 8V\)（某个值）。  
- 再让电压源单独工作：电流源开路。电压源6V加在 \(R_1\) 和 \(R_2\) 串联回路，算出 \(U_2 = 4V\)。  
- 叠加：\(U = 8V + 4V = 12V\)。  
注意功率不能叠加：\(P_{R2} \neq (I_1^2 R_2) + (I_2^2 R_2)\)，因为还有交叉项 \(2I_1 I_2 R_2\)。

## 九、总结

**一句话记住叠加定理**：多个电源共同作用的结果，等于每个电源单独作用（其他电源置零）时结果的总和。  
**三步走**：  
1. 选一个独立源保留，其余电压源短路、电流源开路（受控源不动）。  
2. 计算该源单独作用时某处的电压或电流，标出大小和方向。  
3. 重复对所有独立源，最后把结果代数相加。

**常见错误**：  
- 忘了方向正负，直接加绝对值。  
- 去置零时把受控源也短路或开路了。  
- 用叠加求功率（不允许）。  
- 电路中有非线性元件（比如二极管）还硬用叠加。

# 电路图

我们画一个有**两个电压源**和一个**电流源**的电路，共三个独立源。这样叠加定理的优势就显出来了。

```
             节点A
              /\
             /  \
            /    \
        R1=2Ω    R3=4Ω
          |        |
        10V       I1=2A (向上)
          |        |
          +--------+------- 节点B
          |        |
        R2=3Ω      |
          |        |
         5V        |
          |        |
         GND-------+
```
**两个网孔，三个支路**，每个网孔一个电压源，中间支路一个电流源。

**最终电路描述**：

- 左边一个网孔：10V电池（正极在上），串联电阻 \(R_1=2\Omega\)，然后到达节点A。
- 右边一个网孔：5V电池（正极在上），串联电阻 \(R_2=3\Omega\)，然后到达节点B。
- 在节点A和节点B之间，跨接一个电阻 \(R_3=4\Omega\) 和一个 **2A 电流源**（方向从A指向B，即电流从A流向B）。
- 然后左边网孔从节点A通过 \(R_1\) 回到10V负极（地），右边网孔从节点B通过 \(R_2\) 回到5V负极（地）。地是公共点，所有负极都连在一起。

**画成草图**（横着看）：

```
            R1=2Ω          R3=4Ω        R2=3Ω
  10V (+)---/\/\/---A---/\/\/---B---/\/\/---(+)5V
      (-)                |                 (-)
      GND----------------+-----------------GND
                          
                    I_s = 2A (从A到B)
```

注意：电流源和R3是并联的？不，这里描述的是：A和B之间并联了两个元件：一个R3，一个2A电流源。所以电流源两端电压和R3两端电压相同。

---

# 要求：求电阻 \(R_3\) 上的电流 \(I_{R3}\)（方向假设从A到B）。

---

# 第一步：让10V电压源单独工作

**其他电源置零**：
- 5V电压源 → 短路（用导线代替）
- 2A电流源 → 开路（断开）

此时电路变成：只有10V电池，\(R_1=2\Omega\)，\(R_3=4\Omega\)，\(R_2=3\Omega\)，并且\(R_2\)的右端通过短路线直接连到地（因为5V被短路），左端通过\(R_3\)连到A点。

画一下：10V正极 → \(R_1\) → A点 → 然后有两个去向：
- 经过 \(R_3=4\Omega\) 到 B 点
- 注意：电流源已开路，所以只有\(R_3\)这一条路从A到B
- 从B点经过 \(R_2=3\Omega\) 到地（5V电池短路相当于导线，所以B到地就是\(R_2\)直接到地）
- 地再回到10V负极。

所以整个电路：10V → \(R_1\) → A → \(R_3\) → B → \(R_2\) → 地。这是一个串联回路吗？是的：从10V正极，经过\(R_1\)，再经过\(R_3\)，再经过\(R_2\)，回到负极。注意A和B之间只有\(R_3\)，没有其他支路（因为电流源开路）。总电阻 \(R_1+R_3+R_2 = 2+4+3 = 9\Omega\)。

总电流 \(I_{total} = 10V / 9\Omega \approx 1.111A\)。

这个电流全部流过\(R_3\)，且方向从A到B（因为10V正极在左，电流顺时针）。所以  
\[
I_{R3}^{(1)} = +1.111A
\]

---

# 第二步：让5V电压源单独工作

置零：10V短路，2A电流源开路。

此时电路：只有5V电池（正极在上），串联\(R_2=3\Omega\)，然后到B点，再从B经过\(R_3=4\Omega\)到A点，再从A经过\(R_1=2\Omega\)到地（因为10V短路，A通过\(R_1\)直接到地）。地再回5V负极。

注意：5V正极 → \(R_2\) → B → \(R_3\) → A → \(R_1\) → 地 → 5V负极。这也是一个串联回路。总电阻 \(R_2+R_3+R_1 = 3+4+2=9\Omega\)。

总电流 \(I_{total} = 5V / 9\Omega \approx 0.5556A\)。  
这个电流流过\(R_3\)的方向：从B到A（因为电流从5V正极出来，经过\(R_2\)到B，然后从B到A经过\(R_3\)）。而我们假设的\(I_{R3}\)正方向是从A到B，所以这个电流是反向的。  
\[
I_{R3}^{(2)} = -0.5556A
\]

---

# 第三步：让2A电流源单独工作

置零：10V短路，5V短路。

此时电路：只有2A电流源（从A到B），并联\(R_3=4\Omega\)，还要考虑\(R_1\)和\(R_2\)的连接。因为10V和5V都短路了，A点通过\(R_1\)直接到地，B点通过\(R_2\)直接到地。所以等效电路是：

电流源从A流向B，那么A点比B点电压高。电流源电流2A一部分流过\(R_3\)，一部分流过\(R_1\)和\(R_2\)组成的并联？实际上，A点对地有\(R_1\)，B点对地有\(R_2\)，所以从A到B除了经过\(R_3\)，还有一条路径：A → 地（通过\(R_1\)）→ 地 → B（通过\(R_2\)）？不对，地是同一个节点，所以从A到B经过\(R_1\)和\(R_2\)串联？画出来：

A点：连着\(R_1\)（另一端接地）和\(R_3\)（另一端接B），以及电流源正端（流出）。  
B点：连着\(R_2\)（另一端接地）和\(R_3\)（另一端接A），以及电流源负端（流入）。

所以电流源电流从A流出，分成两路：
- 一路经过\(R_3\)到B
- 另一路经过\(R_1\)到地，然后从地经过\(R_2\)到B（因为地是公共点，所以这一路实际上是从A经过\(R_1\)到地，再从地经过\(R_2\)到B，相当于\(R_1\)和\(R_2\)串联后并联在A、B之间吗？注意：从A直接到B有\(R_3\)，从A到地到B有\(R_1 + R_2\)，所以是并联关系。对！）

所以等效电阻：\(R_{AB} = R_3\) 并联 \((R_1+R_2)\) = \(4 \parallel (2+3) = 4 \parallel 5 = \frac{4\times5}{4+5} = 20/9 \approx 2.222\Omega\)。

电流源总电流2A，按照并联分流：
- 流过\(R_3\)的电流 = \(2A \times \frac{R_1+R_2}{R_3+(R_1+R_2)} = 2 \times \frac{5}{4+5} = 2\times 5/9 = 10/9 \approx 1.111A\)
- 方向：从A到B（因为电流源方向就是从A到B，\(R_3\)与电流源并联，电流方向相同）。

但是注意：电流源单独工作时，它提供的电流2A，其中一部分流过\(R_3\)，另一部分流过\(R_1\)和\(R_2\)串联。我们计算的是\(I_{R3}^{(3)} = 1.111A\)，方向A→B，为正。

等一下，计算一下：2A × (5/(4+5)) = 2×5/9 = 10/9 ≈ 1.111A。正确。

# 第四步：叠加

\[
I_{R3} = I_{R3}^{(1)} + I_{R3}^{(2)} + I_{R3}^{(3)}
= 1.111A + (-0.5556A) + 1.111A
= 1.111 - 0.5556 + 1.111 = 1.6664A
\]

约等于 \(1.666A\)，或写作 \(5/3 A\)（因为 1.111=10/9，0.5556=5/9，10/9 - 5/9 + 10/9 = 15/9 = 5/3）。

所以最终 \(R_3\) 上的电流是 \(5/3 \approx 1.667A\)，方向从A到B。

# 叠加定理计算要点

1. **画清楚电路**，标出每个元件的值和参考方向。  
2. **每次只留一个独立源**，其他独立源置零（电压源短路，电流源开路），受控源保留。  
3. **用欧姆定律、分压分流、串并联化简**等方法算那个源单独作用时的响应。  
4. **注意方向**：如果计算出的电流/电压方向与参考方向一致，取正；反之取负。  
5. **代数相加**得到总响应。  
6. **绝不叠加功率**。
